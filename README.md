# Depression Detection in Dravidian Languages

This repository contains the implementation of our system submitted to the
DravidianLangTech@ACL 2026 Shared Task on Depression Detection.

## Languages
- Tamil
- Malayalam

## Model
- wav2vec 2.0 (fine-tuned)
- SVM baseline with MFCC features

## Evaluation Metric
Macro-F1 Score

## Files
- depression_detection_training.ipynb : Model training and evaluation
- requirements.txt : Python dependencies

## Dataset
The dataset is provided by the DravidianLangTech shared task organizers.

## How to Run
1. Install requirements
2. Run the notebook in Google Colab
